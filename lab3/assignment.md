# Assignment

## Task 1.
**Submit a job which uses an animation from the Blender demo page, for example Repeat Zone - Flower (frame range 1-100). Not all demo scenes work in an environment without display!**

First, let's  download the file

```bash
(base) [ares][plgfranciszekjob@login01 lab3]$ curl -o repeat_zone_flower.blend https://mirrors.dotsrc.org/blender/demo/geometry-nodes/repeat_zone_flower_by_MiRA.blend
```

Then, create the script

```bash
#!/bin/bash
#SBATCH --job-name=blender-job
#SBATCH --output=blender_jpn_%A_%a.out 
#SBATCH -p plgrid
#SBATCH -A plglscclass26-cpu
#SBATCH -N 1
#SBATCH --ntasks-per-node=4
#SBATCH --mem-per-cpu=1GB
#SBATCH --array=1-100

module purge
module load blender

BLEND_FILE="repeat_zone_flower.blend"

OUTPUT_DIR="./blender_output/"
OUTPUT_FORMAT="PNG"

n=${SLURM_ARRAY_TASK_ID}

# Render the animation using Blender CLI
blender -b "$BLEND_FILE" -o "${OUTPUT_DIR}frame_" -F "$OUTPUT_FORMAT" -f "$n"
 
# Notify completion
echo "Rendering completed. Check the output at $OUTPUT_DIR"
```

Run the script
```bash
(base) [ares][plgfranciszekjob@login01 lab3]$ sbatch --array=1-100 assignment.sh
Submitted batch job 19623559
```
